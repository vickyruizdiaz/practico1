¿Còmo preparar las uñas para el esmalte?

Acà te enseñamos a como hacerlo

Materiales que necesitamos:

  • Crema para cuticulas
  • Corta cuticulas
  • Repujador de cuticulas
  • Lima de uñas (100/180)
  • Pulidora de uñas (240)

Preparacion de las uñas:

  1. Agregar crema para cuticulas
  2. Repujar cuticulas
  3. Cortar cuticulas
  4. Limar las uñas y darles forma
  5. Pulir las uñas quitandoles el brillo y la grasa
